# Mobile_Sales_Dashboard – Power BI Project

## Project Overview

This project presents an interactive **Mobile Sales Dashboard** created using **Power BI**.
The dashboard analyzes mobile phone sales data to understand sales performance, customer behavior, and payment trends.

It helps identify key insights such as top-performing brands, sales distribution across cities, customer ratings, and preferred payment methods.

## Project Video
[Watch the Project Video](./Dashboard project.mp4)


## Dashboard Features

* Total Sales, Total Quantity, Transactions, and Average Sales KPIs
* Sales analysis by **City**
* Monthly **Sales Trend Analysis**
* **Customer Rating Distribution**
* **Payment Method Analysis**
* **Brand-wise Sales Performance**
* **Mobile Model Sales Comparison**
* Sales performance by **Day of the Week**
* Interactive filters for **Mobile Model, Payment Method, Brand, and Day**

## Key Insights
* UPI and Credit Card are commonly used payment methods.
* Certain mobile brands contribute significantly to total sales.
* Sales vary across different cities and months.
* Customer ratings show overall satisfaction levels for products.
